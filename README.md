# MATLAB Pill Recognition System💊

A MATLAB-based image processing project for pill recognition and classification.

This project uses computer vision and image processing techniques to identify pills from input images. The system demonstrates basic image preprocessing, feature extraction, and recognition workflows using MATLAB.

---

# Project Overview

The goal of this project is to build a simple pill recognition system that can:

* Read pill images
* Process and enhance image quality
* Detect pill features
* Classify or recognize pills
* Display recognition results

This project is designed for learning image processing and computer vision concepts in MATLAB.

---

# Features

* Image preprocessing
